# COMANS DO  GIT BASH

1 - LS = Lista de pastas .
2 - CD/  + pasta que quero entrar ou  TAB ou seta pra cima pra voltar = Para entrar em alguma pasta
3 - CD.. = Para retornar a pasta anterior
4 - CLS ou Ctrl + L = Para limpar a tela
5 - TAB = Preenche palavras
6 - MKDIR + nome da pasta = Criar pasta 
7 - DEL + nome da pasta = Apaga o conteudo dela
8 - RMDIR = Apaga a pasta toda /S /Q
9 - Seta do Teclado = Navega nos ultimos escritos
10 - Shift + Ins = Colar
11 - LS -A = Listar arquivos ocultos
12 - MV + oque eu quero mover./onde eu quero colocar/ = mover algum arquivo
13 - GIT STATUS = verifica se tem algum arquivo a ser comitado
14 - GIT ADD * = Adiciona todos arquivos para staged
15 - GIT COMMIT -M "resumo das alterações" = Para criar o commit que estava em staged.
16 -  GIT RESTORE --STAGED =
17 - GIT CONFIG --LIST = 
18 - GIT INIT = 
19 - GIT REMOTE ADD + Link no site = Para adicionar nossa árera remota no PC
20 - GIT REMOTE -V = irá listar
21 -  GIT PUSH + arquivo = Para Empurrar do PC para o remoto
22 - GIT PULL + arquivo = Puxar do remoto para ao PC
