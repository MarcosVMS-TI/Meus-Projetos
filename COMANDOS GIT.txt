CMD - ABRIR OS COMANDOS

PRIMEIRO COMANDO: dir enter    aparecer as pastas

SEGUNDO COMANDO: cd/     mais comandos navegar nas pastas

TERCEIRO COMANDO: cd..     para voltar um nivel

QUARTO COMANDO: cls  ou ctrl + L     limpar tudo

QUINTO COMANDO: TAB no teclado     preenche as palavras 

SEXTO COMANDO: mkdir     mais nome da pasta criar pasta

SETMO COMANDO: echo > criar um arquivo .md no final  print do que eu diditei após

OITAVO COMANDO: del    pmais nome da pasta para apagar o conteudo

NONO COMANDO rmdir     para apagar a pasta toda   /S /Q

DECIMO COMANDO: seta do teclado pra cima pra navegar nos comandos

11 - shift + Insert       copia as palavrasls 

12 - ls     listar pastas e arquivos

13 - ls -a       listar arquivos ocultos

14 - mv   + o que eu quero mover ./onde eu quero colocar/

15 - git status     para veririfcar se o arquivo foi modificado pra outra pasta e nao foi comitado

16 - git add   + arquivos ou pastas a serem adicionados

17 - git commit -m "resumo da alteração"     para criar o commit do que eu alterei

18 - git restore --staged     + nome do arquivo

19 - git config --list

20 - git remote add     + o link para adicionar nossa area remota no app 

21 - git remote -v     ira listar

22 - git push     + nome do arquivo para empurar para area remota do git
